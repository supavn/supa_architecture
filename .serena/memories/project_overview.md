## Purpose
Flutter/Dart package providing an opinionated, scalable architecture toolkit (state management, modularization) for Supa apps.

## Tech stack
- Dart 3 / Flutter 3.10+
- Null-safety enabled
- Targets Android, iOS, Web, macOS (plus desktop folders in repo)

## Structure (high level)
- lib/: core library code and blocs
- example/: sample app
- doc/: architecture and design docs
- test/: package tests
- platform folders (android/ios/macos/windows/linux): plugin scaffolding

## Notable docs
- README links to full docs at https://deepwiki.com/supavn/supa_architecture
- Additional markdowns in /doc for architecture, state management, etc.
